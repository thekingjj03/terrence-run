TERRANCE MERCH DASH — VERSION THREE
Kings Cuts Productions x Terrance

HOW TO OPEN
1. Extract this ZIP file.
2. Open index.html in your web browser.
3. No install or internet connection is needed.

CONTROLS
- Move: WASD or Arrow Keys
- Burst: Space Bar
- Touchscreen: drag to move and double-tap to burst

V3 FEATURES
- Full Prototype Series card binder with all 10 supplied card designs.
- Click owned cards in the binder to inspect them full-size and equip their game ability.
- Three playable levels: Prototype Table, Kings Cuts Studio, and School Pop-Up Shop.
- Giant Glue Bottle boss fight: dodge glue and collect Clean-Up Sponges to defeat it.
- Shop and customization: skins plus sunglasses, director cap, mini crown, and gold chain.
- Set completion reward: collect all 10 cards to earn 500 coins and Prototype Master skin.
- In-game How to Play menu/tutorial.
- Card packs and saved browser progress.
- Retains the V2.1 display fix so levels should open at the correct browser size.

NOT INCLUDED YET
- Physical-card code redemption has intentionally been left out for now.

SAVE DATA
The game uses the same browser save key as V2, so existing saved progress should continue when played in the same browser profile.
